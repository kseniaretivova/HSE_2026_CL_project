# HSE_2026_CL_project
## **Палитра современной российской прозы: количественное исследование прилагательных-цветообозначений.**

### **Цель:**
провести количественный анализ особенностей употребления прилагательных‑цветообозначений в произведениях ряда русских авторов XXI века.

### **Задачи:**
1)определить частотность каждого цветообозначения (корпусе на автора);
2)выявить наиболее распространенные цвета (для каждого автора/на корпус в целом), ранжировать их;
*side quest*
3)рассмотреть сочетания “цветообозначение + существительное” и выявить самые частотные сочетания такого типа (для каждого автора/на корпус в целом);
4)выявить наиболее частотные цветообозначения и существительные в составе подобных сочетаний.

### **Материал исследования:**
Корпус из 51 художественного текста российского автора (В.Пелевин, В. Сорокин, М.Елизаров, Г.Яхина, З.Прилепин, А.Иванов, А.Сальников, Е.Водолазкин).

**Критерии отбора текстов:**
- не менее 100 000 слов
- проза
- соответствие жанровому критерию: повесть, роман
- соответствие хронологическому критерию: период публикации - 21 век

**Критерии отбора авторов:**
- соответствие хронологическому критерию: все/значительная часть произведений опубликованы в 21 веке
- личное знакомство с творчеством

### **Этапы работы:**

1)Сбор корпуса

2)Предобработка текстов (приведение к нижнему регистру; удаление лишних символов, знаков препинания (кроме дефисов); замена пробельных символов на обычный пробел); 

3)Поиск и сбор всех прилагательных‑цветообозначений в текстах; извлечение сочетаний «цвет + существительное».

4)Лемматизация

5)Проведение количественного анализа (абсолютная и относительные частоты для цветообозначений + абсолютная частота для сочетаний “цвет + существительное”) и ранжирование результатов.

6)Визуализация (облака слов, тепловая карта, столбчатая диаграмма).

### **Результаты:**
1)Цветообозначения составляют лишь небольшую долю от всех лексических единиц корпуса (для отдельных корпусов по авторам - вывод аналогичен)

2)Наиболее частотные (и абс. ,и отн. частоты) - цветообозначения в корпусе - “белый”, “красный”, “черный”. 

3)Ни одна из палитр авторов не является уникальной (т.е. не содержит цветообозначений, которые не встретились бы у других авторов хотя бы 1 раз).

4) Наибольшие численные значения относительных и абсолютной частоты цветообозначений в корпусах авторов В. Сорокина, В. Пелевина, Г. Яхиной.
 Наименьшие - у Е. Водолазкина.

5)Наиболее частотные словосочетания “цвет + существительное” в корпусе - “”красная жидкость”, “белый свет”,  “красная церемония”, “чёрная смерть”, “белый халат”.

6)Наиболее частотные существительные в составе словосочетаний “цвет + существительное” - свет, цвет, жидкость, небо, халат.
Цветообозначения - красный, белый, чёрный, синий, зелёный.
