# HSE_2026_CL_project
## **Палитра современной российской прозы: количественное исследование прилагательных-цветообозначений.**

### **Цель** - провести количественный анализ особенностей употребления прилагательных‑цветообозначений в произведениях ряда русских авторов XXI века.

### **Задачи:**
1)определить частотность каждого цветообозначения (корпусе на автора);
2)выявить наиболее распространенные цвета (для каждого автора/на корпус в целом), ранжировать их;
*side quest*
3)рассмотреть сочетания “цветообозначение + существительное” и выявить самые частотные сочетания такого типа (для каждого автора/на корпус в целом);
4)выявить наиболее частотные цветообозначения и существительные в составе подобных сочетаний.
